三、判断题（共 8 道）
1. 页岩油气储集层的孔隙度通常低于 10%，需依赖压裂改造才能实现商业开采。（  ）
2. 煤层气的吸附能力与煤阶正相关，高阶煤的吸附容量高于低阶煤。（  ）
3. 储层水敏性越强，说明粘土矿物含量越低，储层越稳定。（  ）
4. 油气资源评价中，容积法适用于勘探早期，地质类比法适用于勘探成熟期。（  ）
5. 凝析气藏的露点压力高于地层压力时，天然气在地下以气态存在，开采时不会析出凝析油。（  ）
6. 镜质体反射率（Ro）时，有机质未成熟，无法生成石油和天然气。（  ）
7. 断层两盘均为砂岩时，断层封闭性好，不易成为油气运移通道。（  ）
8. 油气藏动态分析中，地层压力持续下降且产油量递减，说明油藏进入衰竭开采阶段。（  ）
四、简答题（共 3 道）
1. 简述页岩油气成藏的核心特征及开发关键技术。
2. 简述煤层气与常规天然气在成藏条件、赋存状态及开采方式上的主要差异。
3. 简述油气资源评价的主要流程及各流程的核心任务。
答案解析
一、选择题
1. C  解析：页岩油气储集空间以有机质孔（干酪根演化形成）和微裂缝为主，粒间孔、晶间孔占比极低
，溶蚀孔不发育。
2. C  解析：煤层气主要以吸附相赋存于煤基质表面，占总储量的 80%~90%，游离相和溶解相占比极低
。
3. A  解析：水敏性是指粘土矿物（如蒙脱石）遇水膨胀，堵塞孔隙喉道，导致储层渗透率下降。
4. B  解析：远景资源量是指根据地质条件预测的、尚未证实但可能存在的潜在油气资源量，区别于已探
明的储量。
5. A  解析：凝析气藏露点压力是天然气液化的临界压力，当地层压力低于露点压力时，气态烃会液化析
出凝析油。


## 第4页

6. C  解析：深侧向电阻率测井能反映储层流体性质，含油气储层电阻率显著高于含水储层，是判断含油
气性的核心测井方法。
7. C  解析：镜质体反射率（Ro）是评价有机质成熟度最核心、最常用的指标，不受有机质类型影响，能
精准反映热演化程度。
8. B  解析：孔隙度决定储层储集能力，渗透率决定流体流动能力，二者是油气藏数值模拟中最基础、最
核心的参数。
9. D  解析：断层封闭性取决于岩性对接、断层泥发育、活动时期，与断层长度无直接关联，长断层可能
封闭，短断层也可能开启。
10. A  解析：生油潜力指数 = 氯仿沥青 “A” 含量 / 有机碳含量，反映有机质向油气转化的效率，比值越高
生油潜力越好。
11. C  解析：油气运移优先选择连通性好、阻力小的通道，裂缝与高渗砂体满足这一条件，其他选项通道
连通性差或阻力大。
12. C  解析：含气饱和度指有效孔隙中天然气体积与有效孔隙体积之比，仅考虑能参与流动的有效孔隙，
排除孤立孔隙。
13. B  解析：圈闭容量是指圈闭能够容纳的油气总体积，取决于圈闭面积、有效厚度和有效孔隙度。
14. A  解析：浅海深水陆棚相页岩厚度大、有机质含量高、脆性矿物丰富，是页岩油气发育的最有利沉积
相。
15. C  解析：地层压力变化直接反映油藏能量消耗与驱动类型，如水压驱动压力下降缓慢，弹性驱动压力
下降较快。
二、填空题
1. 高有机质含量；高孔隙度  解析：页岩油气成藏需满足高有机质（生烃物质基础）、高孔隙度（储集空
间）和高脆性矿物（可压裂性）三大核心条件。
2. 生物化学作用；吸附富集  解析：煤层气成藏先经生物化学作用生成天然气，再通过煤基质吸附富集，
最后在压力差作用下渗流运移。
3. 碱敏性；水敏性  解析：储层敏感性包括水敏、盐敏、酸敏、速敏、碱敏五类，水敏性因粘土矿物膨胀
普遍存在，是最常见的储层伤害类型。
4. 统计分析法；数值模拟法  解析：油气资源评价方法包括地质类比法、容积法、统计分析法、数值模拟
法，分别适用于不同勘探阶段和数据条件。
5. 高温高压；特定烃类组成  解析：凝析气藏需在高温高压条件下，天然气中重烃组分才能保持气态，特
定烃类组成（含一定重烃）是形成凝析油的前提。
6. 0.5%；1.3%  解析：Ro=0.5% 时有机质进入生油窗，开始大量生成石油；Ro=1.3% 时进入高成熟阶段
，石油热裂解为主，主要生成天然气。


## 第5页

7. 岩性封闭；力学封闭  解析：断层封闭性机制包括岩性封闭（两盘岩性差异导致遮挡）和力学封闭（断
层泥胶结、压实形成封闭）。
8. 产油量；含油饱和度  解析：油气藏动态分析核心参数包括地层压力、产油量、含油饱和度、含水率，
可全面判断开发阶段、能量变化及剩余潜力。
三、判断题
1. √  解析：页岩油气储集层孔隙度通常 5%~10%，渗透率 mD，天然产能极低，必须通过水平井体积压
裂改造才能实现商业开采。
2. √  解析：煤阶越高，煤基质比表面积越大，吸附容量越强，高阶烟煤和无烟煤的煤层气吸附量显著高
于低阶褐煤。
3. ×  解析：水敏性越强，说明粘土矿物（尤其是蒙脱石）含量越高，储层遇水越易膨胀伤害，稳定性越
差。
4. ×  解析：容积法适用于勘探中期（有钻井资料），地质类比法适用于勘探早期（无钻井资料，类比已
知盆地），统计分析法适用于成熟期。
5. √  解析：凝析气藏地层压力高于露点压力时，天然气保持气态；低于露点压力时才会液化析出凝析油
，开采时需控制压力避免相变。
6. √  解析：Ro<0.5% 时有机质处于未成熟阶段，仅能生成生物气或未熟油，无法形成工业价值的石油和
天然气。
7. ×  解析：断层两盘均为砂岩时，岩性差异小，易形成开启性断层，成为油气运移通道；两盘为砂泥岩
对接时封闭性更好。
8. √  解析：地层压力持续下降、产油量递减是油藏衰竭开采阶段的典型特征，表明天然能量逐渐消耗，
需人工补充能量（如注水）维持产量。
四、简答题
1. 解析：页岩油气成藏核心特征：①生储一体，页岩既是生油岩也是储集层；②低孔低渗（孔隙度 5%~
10%、渗透率 < 0.1mD），依赖有机质孔和微裂缝；③连续型分布，无明显圈闭边界；④吸附相和游
离相共存。开发关键技术：①水平井钻井技术（扩大储层接触面积）；②体积压裂技术（形成复杂裂
缝网络，改善渗流）；③页岩含油气性评价技术（精准识别甜点区）；④分段压裂与裂缝监测技术（
确保改造效果）。
2. 解析：主要差异：①成藏条件：煤层气成藏依赖煤储层，需高煤阶、高有机质含量；常规天然气成藏
依赖砂岩 / 碳酸盐岩储层，需生储盖组合与圈闭。②赋存状态：煤层气以吸附相为主（80%~90%）；
常规天然气以游离相为主。③开采方式：煤层气需先排水降压，使吸附气解吸后开采；常规天然气通
过井底压力差直接开采，无需排水降压步骤。


## 第6页

3. 解析：油气资源评价主要流程及核心任务：①区域地质研究（核心任务：分析盆地构造、沉积、烃源
岩发育特征，明确勘探潜力区）；②烃源岩评价（核心任务：评价有机质丰度、类型、成熟度，计算
生烃量）；③储集层与圈闭评价（核心任务：评价储层孔渗性、圈闭有效性，估算圈闭容量）；④资
源量计算（核心任务：选用合适方法（容积法、类比法等）计算远景资源量、控制资源量、探明储量
）；⑤资源综合评价（核心任务：分析资源分布、勘探风险与经济效益，提出勘探部署建议）。
