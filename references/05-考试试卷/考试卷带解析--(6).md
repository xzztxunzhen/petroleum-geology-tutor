四、简答题（共 3 道）
1. 简述油气藏压力系统的类型及各类型的形成机制。
2. 简述生油岩评价的核心指标体系及各指标的地质意义。
3. 简述致密气与页岩气在成藏特征及开发技术上的核心差异。
答案解析
一、选择题
1. B  解析：Ro=0.5%~1.3% 是有机质成熟阶段，即生油窗，大量生成石油；为未成熟，1.3%~2.0% 为高
成熟，>2.0% 为过成熟。
2. C  解析：致密气储层以晶间孔隙为主，粒间孔隙和溶蚀孔隙占比低，有机质孔是页岩气的主要孔隙类
型。
3. A  解析：异常低压的压力系数 <0.8，正常压力 0.8~1.2，异常高压> 1.2，异常低压多由地层抬升剥蚀或
地下水流失导致。
4. C  解析：地震勘探中的 “亮点反射” 是油气藏的典型特征，由油气与地层水的波阻抗差异形成，可直
接识别潜在油气藏。
5. B  解析：断层 - 岩性圈闭由断层遮挡和岩性变化共同形成，属于岩性 - 构造复合圈闭，其他选项为单一
类型圈闭。
6. A  解析：孔隙度的定义是孔隙体积与岩石总体积之比，有效孔隙度的分子是有效孔隙体积。
7. B  解析：毛细管力是油气运移的最主要阻力，尤其在细粒岩石中，需克服毛细管力才能实现油气运移
。
8. B  解析：有效盖层的厚度通常不小于 10m，厚度过小易产生裂缝，影响封闭性，泥岩、膏盐岩盖层厚
度达标后封盖效果更佳。
9. C  解析：深水重力流相砂体埋藏深、压实作用强，易形成致密储层，是致密气发育的有利沉积相。


## 第4页

10. C  解析：统计分析法通过建立地质参数（如储层厚度、孔隙度）与资源量的统计关系，预测未知区域
资源量，适用于勘探中期。
11. D  解析：绿泥石稳定性最强，遇水不易膨胀或迁移，对储层渗透率伤害最小；蒙脱石稳定性最差，水
敏性最强。
12. D  解析：可燃冰形成需低温（0~10℃）、高压（>10MPa）和充足甲烷来源，与储层孔隙度无必然关
联，在孔隙度较低的沉积物中也可形成。
13. A  解析：圈闭需在油气生排烃高峰期前形成，才能有效聚集油气，形成时间晚于高峰期则无法捕获油
气，圈闭无效。
14. B  解析：油藏含水率升高的主要原因是边水锥进或底水上升，随着开采进行，边底水侵入油井，导致
含水率上升。
15. B  解析：页岩气成藏的关键保存条件是构造稳定，避免断层破坏导致油气逸散；厚层泥岩盖层、异常
高压是辅助条件，高脆性矿物含量影响可压裂性。
二、填空题
1. 异常低压（<0.8）；异常低压  解析：油气藏压力系数按数值分为三类，异常低压多由地层抬升剥蚀导
致压力释放，或地下水流失引起压力降低。
2. 高成熟；过成熟  解析：生油岩成熟度四阶段对应产物：未成熟（生物气）、成熟（石油）、高成熟（
凝析气）、过成熟（干气）。
3. 高有机质丰度；低孔低渗储层  解析：致密气成藏需高有机质丰度提供油气来源，低孔低渗储层作为储
集空间，配合有效保存条件形成商业气藏。
4. 流体粘度阻力；岩石吸附阻力  解析：油气运移阻力包括毛细管力、流体粘度阻力（流体流动自身粘性
）和岩石吸附阻力（有机质对油气的吸附），三者共同影响运移效率。
5. 孔隙度；渗透率  解析：盖层有效性评价核心是厚度、孔隙度和渗透率，低孔低渗且厚度达标的盖层封
闭性最强。
6. 储层发育；生油岩分布  解析：沉积相控制储层的岩性、厚度、孔渗性（储层发育），同时控制生油岩
的分布范围和有机质丰度，间接影响油气来源。
7. 含油（气）饱和度；含油（气）面积  解析：油气资源量计算需核心参数：储层厚度、孔隙度、含油（
气）饱和度、含油（气）面积，通过容积法即可估算资源量。
8. 体积压裂；水平井分段压裂  解析：非常规油气开发依赖水平井扩大储层接触面积，体积压裂和分段压
裂技术形成复杂裂缝网络，改善低渗储层的渗流条件。

√  解析：Ro=1.3%~2.0% 为高成熟阶段，有机质热裂解作用增强，石油转化为凝析气和天然气，Ro>

2. ×  解析：致密气储层的核心特征是低孔低渗，孔隙度通常小于 10%，渗透率小于 0.1mD，需压裂改造
才能实现商业开采。
3. √  解析：断层两盘 “砂 - 泥对接” 时，泥岩的低孔低渗特性形成遮挡，断层封闭性好，利于油气聚集
；“砂 - 砂对接” 时封闭性差，易成为运移通道。
4. ×  解析：盖层封闭性不仅取决于厚度，还与岩石类型（如膏盐岩封闭性优于泥岩）、孔隙度、渗透率
相关，薄层膏盐岩的封闭性可能优于厚层疏松泥岩。
5. √  解析：油气运移中，游离相占主导地位，溶解相（溶于地层水）和扩散相（分子扩散）仅在特定条
件下（如高压、低粘度）存在，占比极低。
6. √  解析：非常规油气（页岩气、致密气、煤层气）呈连续型分布，无明确圈闭边界，区别于常规油气
的离散型圈闭聚集特征。
7. √  解析：正常压力油气藏压力随埋藏深度线性升高（梯度约 10MPa/1000m），异常高压或低压因构
造挤压、欠压实、抬升剥蚀等因素，不符合线性规律。
8. √  解析：油气资源量是地下潜在油气总量（含未探明部分），储量是资源量中经钻探证实、在当前技
术经济条件下可开采的部分，二者是包含与被包含关系。
四、简答题
1. 解析：油气藏压力系统分为正常压力系统、异常高压系统和异常低压系统。形成机制：①正常压力系
统：由静水柱压力控制，压力随深度线性升高，无特殊地质作用影响；②异常高压系统：主要由欠压
实作用（沉积物快速埋藏，孔隙水无法排出）、构造挤压、流体热膨胀或矿物转化释水形成；③异常
低压系统：多由地层抬升剥蚀（压力释放）、地下水流失（孔隙水减少导致压力降低）或构造拉伸作
用形成。
2. 解析：生油岩评价核心指标体系包括有机质丰度、有机质类型和有机质成熟度。地质意义：①有机质
丰度（有机碳含量、氯仿沥青 “A” 含量）：反映生油岩生烃物质基础，含量越高生烃潜力越大；②
有机质类型（腐泥型、混合型、腐殖型）：决定生烃产物类型，腐泥型以生油为主，腐殖型以生气为
主；③有机质成熟度（镜质体反射率 Ro、生烃潜量指数）：反映有机质热演化程度，决定生烃阶段和
油气品质，Ro 处于 0.5%~1.3% 时生油效率最高。
3. 解析：核心差异：①成藏特征：致密气储层为砂岩或碳酸盐岩，以游离相为主，需常规圈闭配合；页
岩气储层为页岩，生储一体，以吸附相和游离相共存，无明确圈闭边界。②开发技术：致密气开发以
水平井 + 体积压裂为主，重点改善储层连通性；页岩气开发需水平井 + 分段压裂，同时注重页岩脆性
矿物含量（影响压裂效果），部分区块需排水降压促进吸附气解吸。
